What is thfProject

A Void World

No ground. No rules. An infinite void stretches in every direction. Every 30 seconds a random item drops into your inventory — and vanilla structures spawn right in the void, so you can find End cities, Nether fortresses, strongholds and more floating in the dark.

Core Features

How It Works

01 VOID

Infinite Void

No floor. No ceiling. No walls. An endless black expanse in every direction. Every block you stand on is one you placed — or found. Falling ends everything.

02 ITEM

Random Items

Every 30 seconds the void delivers a random item to your inventory. Diamond sword? Gravel? Lava bucket? You adapt to what fate sends.

03 STRUC

Vanilla Structures

It's a void world — but vanilla structures still spawn. Find End cities, Nether fortresses, strongholds, dungeons and more, all floating in the darkness with nothing beneath them.

Server Rules

Semi-Anarchy

THF runs on a single rule so play stays fair without becoming restrictive. Raid, grief, exploit bugs, use autoclickers, build, or destroy — nearly everything is on the table. The one line is client software that automates combat or bypasses server-side game mechanics. If the server cannot detect it and it hands you an edge in a fight, it is banned.

Allowed
  • Bug and glitch exploitation, including item duplication methods
  • Autoclickers and keyboard macros for any in-game action
  • Griefing, raiding, and unrestricted PvP with no boundaries
  • Teaming, alliances, and betrayal — social dynamics are yours to manage
  • Performance and QoL mods — Sodium, OptiFine, Iris, Lithium, and equivalents
  • Any vanilla game mechanic used creatively or destructively
Banned
  • Kill aura and combat reach modifications — automated targeting or extended hitboxes
  • Fly hacks and speed hacks outside server-permitted flight
  • X-ray, ESP, or any client that reveals hidden terrain or player positions through walls
  • Any modification that manipulates server-side combat, movement, or game state in ways impossible through normal input
